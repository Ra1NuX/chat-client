
# Instalacion : 

Requisitos:   
- Tener Node instalado.

Para usar el programa: 
iniciar un PowerShell desde la carpeta descargada.
usar el comando:

    npm i 
    
para instalar las dependencias.
y usar el comando:

    npm start

para iniciar el programa.
